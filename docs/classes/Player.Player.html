    Class Player

    Vreo 播放器主类

    基于如视三维渲染引擎 Five 和 React 实现的 3D 空间剧本播放器。 支持播放包含相机运动、特效、标签等多种关键帧的 VR 视频内容。

    Player

    Example

    import { Player } from '@realsee/vreo';
    import { Five } from '@realsee/five';

    // 创建 Five 实例
    const five = new Five({ /* Five 配置 * / });

    // 创建播放器实例
    const player = new Player(five, {
      container: document.getElementById('player-container'),
      keyframeMap: { /* 关键帧配置 * / },
      autoPreload: true
    });

    // 监听播放事件
    player.on('playing', () => console.log('开始播放'));
    player.on('paused', () => console.log('暂停播放'));

    // 加载并播放 VR 视频
    player.load(vreoUnit).then(() => {
      console.log('加载完成');
    });

    load

    • load(
          vreoUnit: VreoUnit,
          currentTime?: number,
          preload?: boolean,
          force?: boolean,
      ): Promise<boolean>

      加载并播放 VR 视频内容

      这是播放器的核心方法，负责加载 VreoUnit 数据并开始播放。 包括预处理、预加载、事件设置等完整的加载流程。

      Parameters

      • vreoUnit: VreoUnit

        VR 视频单元数据，包含关键帧、视频信息等

      • currentTime: number = 0

        开始播放的时间点（毫秒），默认为 0

      • preload: boolean = false

        是否预加载点位数据，默认为 false

      • force: boolean = false

        是否强制重新创建数据副本，默认为 false

      Returns Promise<boolean>

      Promise<boolean> - 加载完成后返回 true

      Fires

      VreoKeyframeEvent#loaded - 数据加载完成时触发

      Example

      // 基本加载
      await player.load(vreoUnit);

      // 从指定时间开始播放
      await player.load(vreoUnit, 5000); // 从第5秒开始

      // 启用预加载
      await player.load(vreoUnit, 0, true);

    play

    • play(currentTime?: number): boolean

      开始或继续播放

      这个方法处理播放器的播放控制逻辑，包括：

      • 播放状态检查和防重复播放
      • 播放时间控制
      • 背景音乐关键帧的重置逻辑
      • 调试接口暴露

      Parameters

      • OptionalcurrentTime: number

        可选的播放起始时间（毫秒）

      Returns boolean

      操作是否成功

      Fires

      VreoKeyframeEvent#playing - 开始播放时触发

      Example

      // 开始播放
      player.play();

      // 从指定时间开始播放
      player.play(10000); // 从第10秒开始
