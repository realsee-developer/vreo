    Optionalpreload

    preload?: boolean

    是否开启音频预载能力。仅对.mp3有效。

    Description

    开启预载能力后会通过 Fetch/XHR 方式将音频文件下载转成二进制 Blob 提供给多媒体实例。这样的好处是 能够保障音视频播放过程中不卡顿。 但在部分 iOS 系统中会存在兼容问题——播 25s 之后会静音（muted 为 false，但是没有声音）。
